This is the lazy avoidance of git repo trickery or packaging up jovlinger/bin into something that can live in requirements.txt.

Copied artifacts include ``venv-run`` / ``venv-run.py`` (Python shebang interpreter for utils CLIs), ``run-with-stdout-logged.py``, and ``mock_cmd.py``.
